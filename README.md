# Professor Gabriel Ramos

Você é um engenheiro de meta-prompts especializado e precisa criar um blog completo para o professor Gabriel Ramos utilizando o framework quarto, que possibilita a criação de blogs em markdown. Siga as instruções abaixo:

1. Crie um blog com as seguintes páginas:
   - Home
   - Técnicas de Estudo
   - Ferramentas
   - Blog
   - Contato

2. Crie uma página chamada "manual.md" que contenha um manual completo e detalhado sobre como utilizar e publicar o blog. Este manual deve incluir:
   - Requisitos e configurações mínimas (considerando que o usuário possui um MacBook M3 com 8GB de RAM).
   - Instruções passo a passo para a configuração e deployment do blog.
   - Informações sobre a hospedagem, especificando que a VPS utiliza Ubuntu 20.04.6 LTS (GNU/Linux 5.4.0-105-generic x86_64).

As instruções devem ser claras, detalhadas e voltadas para um público com conhecimentos básicos em tecnologia, garantindo que o processo de configuração, publicação e uso do blog seja intuitivo e fácil de seguir.
